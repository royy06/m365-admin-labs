## Microsoft 365 Admin Labs

This repository documents hands-on Microsoft 365 administration learning through structured, scenario-based practice.

The goal is to build practical understanding of identity, security, licensing, and administration tasks commonly used in enterprise environments.

## What This Covers
- Microsoft Entra ID (users, groups, roles)
- Microsoft 365 licensing (E1, E3, E5)
- Identity lifecycle management
- Access control fundamentals
- Security and troubleshooting scenarios

## Structure

### 01-Identity-Lifecycle
Core identity management including user onboarding, groups vs roles, and licensing.

### 02-Access-Control
MFA and Conditional Access fundamentals.

### 03-Collaboration
SharePoint, Teams, and access management.

### 04-Exchange-Recovery
Mailbox basics, mail flow, and recovery scenarios.

### 05-Monitoring-Troubleshooting
Sign-in logs, audit logs, and troubleshooting methods.

### 06-Compliance
Retention and DLP policies.

### 07-Intune-Basics
Device enrollment and compliance policies.

### 08-Admin-Playbook
Real-world runbooks for onboarding, troubleshooting, and security.

## Goal
To demonstrate practical Microsoft 365 administration skills through documentation and scenario-based learning.
