# User Onboarding in Microsoft 365

## Overview
User onboarding is the process of creating and preparing a new user account in Microsoft 365 with the correct access, permissions, and licensing.

## Steps
1. Create user in Microsoft Entra ID
2. Assign username and temporary password
3. Add user to required security or Microsoft 365 groups
4. Assign appropriate license (E1, E3, or E5)
5. Force password change at first login
6. Verify access to Teams, Outlook, and SharePoint

## Why It Matters
Proper onboarding ensures users have the correct access on day one while maintaining security and least privilege principles.
