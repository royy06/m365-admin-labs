# Groups vs Roles in Microsoft 365

## Groups
Groups are used to manage access to resources such as Teams, SharePoint, and licenses.

Examples:
- Microsoft 365 Groups
- Security Groups
- Distribution Lists

## Roles
Roles are used to assign administrative permissions in Microsoft 365.

Examples:
- Global Administrator
- User Administrator
- Helpdesk Administrator

## Key Difference
Groups control access to resources.
Roles control administrative capabilities.

## Best Practice
Use groups for user access and roles only when administrative permissions are required to reduce security risk.
