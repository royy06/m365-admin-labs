# Microsoft 365 License Assignment

## Overview
Licensing determines what services a user can access in Microsoft 365.

## Assignment Methods
- Direct assignment to user
- Group-based licensing (recommended)

## Process
1. Identify user job role
2. Select appropriate license (E1, E3, E5)
3. Assign license in Microsoft 365 Admin Center
4. Verify services are enabled

## License Removal
- Remove license when user leaves the organization
- Prevents unnecessary cost and protects data access
