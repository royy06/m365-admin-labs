# SharePoint Permissions

## Overview
SharePoint permissions control who can view, edit, or manage content within SharePoint sites.

## Permission Levels
- Owners – Full control
- Members – Edit access
- Visitors – Read-only access

## Best Practices
- Assign permissions through groups, not individual users
- Use least privilege
- Avoid breaking inheritance unless necessary

## Common Issues
- User added but still denied access
- Permissions inheritance broken incorrectly
