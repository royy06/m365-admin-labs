# Troubleshooting Framework

## Step 1: Identify the issue
Understand what the user is reporting.

## Step 2: Gather information
- User account details
- Error messages
- Time of issue
- Affected service

## Step 3: Check logs
- Sign-in logs (authentication issues)
- Audit logs (activity issues)

## Step 4: Identify root cause
Determine what is actually causing the issue.

## Step 5: Apply fix
- Reset password
- Reassign license
- Adjust permissions
- Modify policy

## Step 6: Verify resolution
Confirm user can access the service again.
