# Scenario: User Lost Access to Microsoft 365 Apps

## Issue
User cannot access Outlook, Teams, or SharePoint.

## Possible Causes
- License removed
- License expired
- Incorrect license assignment

## Troubleshooting Steps
1. Check assigned licenses in Admin Center
2. Verify subscription status
3. Reassign correct license
4. Confirm service activation

## Resolution
Reassign Microsoft 365 license and verify access restoration.
