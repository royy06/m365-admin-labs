# Microsoft Entra Sign-in Logs

## What are Sign-in Logs
Sign-in logs track authentication attempts for users accessing Microsoft 365 services.

## What they show
- Successful and failed login attempts
- IP address and location
- Device information
- Authentication method used

## Why they are important
Used to detect:
- Suspicious activity
- Password attacks
- Account lockouts

## Where to access
Microsoft Entra ID → Monitoring → Sign-in logs
