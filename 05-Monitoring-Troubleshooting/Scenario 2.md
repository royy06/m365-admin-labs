# Scenario: User Cannot Access SharePoint

## Issue
User receives “Access Denied” when opening SharePoint site.

## Possible Causes
- Missing permissions
- Incorrect group membership
- Conditional Access policy restriction

## Troubleshooting Steps
1. Check SharePoint permissions
2. Verify group membership
3. Review Conditional Access policies
4. Confirm correct site URL

## Resolution
Add user to correct SharePoint group and restore access.
