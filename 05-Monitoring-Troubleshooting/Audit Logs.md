# Microsoft Purview Audit Logs

## What are Audit Logs
Audit logs record user and admin activity across Microsoft 365 services.

## Examples of tracked activity
- File access and deletion
- User creation or modification
- Permission changes
- Email actions

## Why they matter
They help administrators:
- Investigate security incidents
- Track user activity
- Ensure compliance

## Where to access
Microsoft Purview → Audit → Search
