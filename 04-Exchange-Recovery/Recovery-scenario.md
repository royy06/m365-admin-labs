# Scenario: User Deleted Important Email

## Issue
User reports that an important email was accidentally deleted.

## Troubleshooting Steps
1. Check Deleted Items folder
2. Check “Recoverable Items” folder
3. Use Microsoft 365 Admin Center if needed
4. Verify retention policy settings

## Resolution
Restore email from Recoverable Items or Deleted Items depending on time elapsed.

# Scenario: User Mailbox Missing

## Issue
User cannot access mailbox after login.

## Possible Causes
- License removed
- Account disabled
- Provisioning delay

## Resolution
Reassign license and verify mailbox status in Exchange Admin Center
