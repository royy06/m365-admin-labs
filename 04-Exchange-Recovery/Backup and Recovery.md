# Backup and Recovery in Microsoft 365

## Overview
Microsoft 365 uses built-in retention policies rather than traditional backups.

## Recovery Options
- Deleted Items folder (user recovery)
- Recoverable Items folder (admin recovery)
- SharePoint Recycle Bin
- OneDrive version history

## Common Recovery Scenarios
- Deleted email recovery
- Restoring deleted SharePoint files
- Recovering user mailbox after license removal

## Key Point
Microsoft 365 provides retention-based recovery, not traditional backup systems.
