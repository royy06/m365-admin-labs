# Mailbox Lifecycle in Microsoft Exchange Online

## Overview
A mailbox represents a user's email account in Microsoft 365 Exchange Online.

## Lifecycle Stages
1. Creation – mailbox is created when a license is assigned
2. Active use – user sends and receives emails
3. Disabled – license removed or user account disabled
4. Soft deleted – mailbox retained for a short period
5. Permanently deleted – removed after retention period

## Key Point
Mailbox existence is directly tied to the user’s Microsoft 365 license.
